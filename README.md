# Supermarket-App
A supermarket shelves layout dynamically generated

## Install

Clone this repository and run:

```
npm i
```

## Run

### Dev

In the first terminal, run:

```
npm run dev:server
```

In a second terminal, run:

```
npm run dev:client
```

### Production

First, build the project:

```
npm run build:client
```

Then run:

```
npm run prod:server
```
